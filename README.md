# LawBuddy: Indian Legal Literacy Matrix Terminal

LawBuddy is an open-source, ultra-fast, zero-dependency desktop terminal engineered to democratize Indian legal literacy. Built using native Python Tkinter structures, it bypasses heavy modern web frameworks to run smoothly rendering a completely sharp, non-pixelated 3D vector neural mesh.

---

## Key Technical Core Features

* **3D Vector Synapse Mesh Engine:** Custom isometric matrix engine projecting dynamic legal modules ($X, Y, Z$) with zero external graphics library overhead.
* **Instant Indian Legal Mapping:** Fine-tuned to translate complex laws—including the Bharatiya Nyaya Sanhita (BNS), Consumer Protection Act, and IT Act—into highly scannable, clear documentation.
* **Enterprise Firewall Immune:** Structured using native Windows processes to bypass Smart App Control and corporate Device Guard limitations seamlessly.
